# Pylette
Simple Python GUI app for creating palletes of RGB colors.

## Features:
- [ ] multiple palettes
- [ ] palette autoscaling
- [ ] intuitive clipboard integration
- [ ] export to text file
- [ ] export to .png image
- [ ] autosave
